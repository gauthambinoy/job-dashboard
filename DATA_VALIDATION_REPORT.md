### 5.1 Matching Algorithm

| **Salary Match** | 15% | Range overlap between user expectations and job offer |
| **Location Match** | 10% | Exact country match |
| **Education Match** | 5% | Degree requirement satisfaction |

**Skills Match (40% weight):**
```typescript
function calculateSkillsMatch(userSkills, requiredSkills) {
  if (requiredSkills.length === 0) return 1;
  
  const matchedCount = requiredSkills.filter(skill =>
    userSkills.some(uSkill =>

    )
```

**Result:** ✅ WORKING CORRECTLY
- Fuzzy matching allows partial skill name matches
- Score normalized to 0-1 range
- Properly weighted in final calculation

**Experience Match (30% weight):**
```typescript
if (jobExpMatch.includes('junior') || jobExpMatch.includes('entry')) {
  return userExp <= 2 ? 1 : 0.7;
```

3. **Average Match Score:**
```sql
SELECT AVG(match_score) as avg_score
FROM jobs;
```
Expected range: 50-75% (depends on user profile)

4. **Match Score Distribution:**
```sql
SELECT 
  CASE 
    WHEN match_score <= 20 THEN '0-20'
    WHEN match_score <= 40 THEN '20-40'
    WHEN match_score <= 60 THEN '40-60'
    WHEN match_score <= 80 THEN '60-80'
    ELSE '80-100'
  END as score_range,
  COUNT(*) as count
FROM jobs
GROUP BY score_range;
```

5. **Cluster Statistics:**
```sql
SELECT 
  jc.domain,
  COUNT(DISTINCT jc.id) as cluster_count,
  AVG(jc.avg_match_score) as avg_cluster_match
FROM job_clusters jc
GROUP BY jc.domain;
```

6. **Top Skills Frequency:**
```sql
SELECT skill, COUNT(*) as frequency
FROM (
  SELECT UNNEST(extracted_skills_required) as skill FROM jobs
) t
GROUP BY skill
ORDER BY frequency DESC
LIMIT 10;
```
