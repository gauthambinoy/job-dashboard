ROUTES AND PAGES:
=================

/                          Home landing page with feature overview
                          File: app/page.tsx

/profile                   User profile setup and preferences
                          - Skills multi-select
                          - Experience and education
                          - Salary range
                          - Target countries
                          - Availability status
                          File: app/profile/page.tsx

/search                    Job search and filter interface
                          - Domain selection
                          - Country filter
                          - Experience level
                          - Salary range sliders
                          - Availability filter
                          File: app/search/page.tsx

/search (results view)     Search results table display
                          - Company, Title, Location
                          - Salary, Match %, Cluster ID
                          - Pagination support
                          File: app/search/results.tsx (conditional render)

/jobs/[id]                 Individual job details page
                          - Full job description
                          - Requirements and skills
                          - Match breakdown chart
                          - Cluster information
                          - Apply button
                          File: app/jobs/[id]/page.tsx

/tracker                   Application tracking dashboard
                          - Status overview cards
                          - Applications table
                          - Editable status
                          - Notes and comments
                          - Filter and sort
                          File: app/tracker/page.tsx

/analytics                 Analytics and insights
                          - Dashboard metrics
                          - Charts and visualizations
                          - Cluster analysis
                          File: app/analytics/page.tsx

/analytics/clusters        Cluster detailed analysis
                          File: app/analytics/clusters/page.tsx (existing)

  Used in: app/tracker/page.tsx, app/analytics/page.tsx

  Used in: app/profile/page.tsx, app/jobs/[id]/page.tsx

  Used in: app/analytics/page.tsx
  Props: stages[], className?
